# Problem Statement & Project Scope

## Problem Statement
Create a quiz game based on the KBC format with randomized questions and safe levels for entertainment.

## Scope of the Project
- Console-based program
- Random question ordering
- Option selection via input
- Two milestone safe levels

## Target Users
- Beginners learning Python
- Students needing small academic projects
- Anyone who enjoys quiz-style games

## High-Level Uses
- Demonstrates Python input/output, loops, lists, and randomization
- Helps understand more about a sport
