# KBC Quiz Game 

## Overview
This is a Python-based quiz game inspired by *Kaun Banega Crorepati (KBC)*. 
Questions appear randomly, players choose an option (1–4), and safe levels protect winnings.

## Features
- Randomized questions
- Four-option multiple-choice format
- Safe-level winnings (10,000 and 3,20,000)
- Option to quit (enter 0)
- Simple console-based interface

## Steps to Install
1. Install Python 3 (if not already installed)
2. Save the script as `kbc_quiz.py`
3. Open terminal and run:
   ```
   python kbc_quiz.py
   ```

## Instructions for Testing
1. Run the program.
2. Answer questions by entering 1–4.
3. Enter `0` to quit anytime.
4. Test safe levels by reaching question 5 and 10.
5. Test wrong answers to verify safe-level fallback.
